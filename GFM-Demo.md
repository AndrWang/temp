|  函数       |      说明        |
| ----------- | ---------------- |
| Left()      | 返回串左边的函数 |
| Right()     | 返回串右边的字符 |
| Length()    | 返回串的长度     |
| Locate()    | 找出串的一个子串 |
| Lower()     | 将串返回小写     |
| Upper()     | 将串转换为大写   |
| LTrim()     | 去掉串左边的空格 |
| RTrim()     | 去掉串右边的空格 |
| Soundex()   | 返回串的SOUNDEX值|
| SubString() | 返回子串的字符   |
